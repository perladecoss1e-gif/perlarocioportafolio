🌐 Lenguajes de Desarrollo Web

Son los lenguajes usados para construir sitios y aplicaciones web. Se dividen en front-end (lo que el usuario ve) y back-end (el servidor y la lógica de negocio). Todo lo que ves en tu navegador fue escrito con alguno de estos lenguajes.

HTML

HTML 5

HyperText Markup Language

Es el esqueleto de toda página web. Define la estructura y el contenido: títulos, párrafos, imágenes, enlaces y formularios. No es un lenguaje de programación, sino de marcado.

index.html 
<!-- Estructura básica HTML -->
<!DOCTYPE html>
<html lang="es">
  <head>
    <title>Mi Primera Página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
    <p>Aprendo <strong>HTML</strong> 🚀</p>
  </body>
</html>
Front-endEstructuraWeb
CSS

CSS 3

Cascading Style Sheets

Le da estilo y apariencia al HTML. Con CSS controlamos colores, fuentes, tamaños, animaciones y el diseño general de la página. Es el "maquillaje" de la web.

estilos.css 
/* Estilos para el cuerpo */
body {
  background: linear-gradient(135deg,
    #fce4ec, #ede0ff);
  font-family: 'Raleway', sans-serif;
  color: #1a1a2e;
}

h1 {
  font-size: 2.5rem;
  color: #e91e8c;
  animation: fadeIn .8s ease;
}
Front-endEstilosAnimaciones
JS

JavaScript

Lenguaje de scripting web

Agrega interactividad y lógica a las páginas web. Con JS podemos crear menús, animaciones, validar formularios, consumir APIs y construir aplicaciones completas.

app.js 
// Función de saludo personalizado
const saludar = (nombre) => {
  const hora = new Date().getHours();
  const saludo = hora < 12
    ? "Buenos días"
    : "Buenas tardes";
  return `${saludo}, ${nombre}!`;
};

console.log(saludar("Perla"));
// → "Buenas tardes, Perla!"

Python

Back-end & IA

PHP

PHP

Hypertext Preprocessor

Lenguaje de servidor que impulsa el 77% de la web, incluyendo WordPress. Genera HTML dinámico, maneja bases de datos y sesiones de usuario. Es ideal para principiantes en back-end.

index.php 
<?php // Script de bienvenida
$nombre = "Perla";
$fecha  = date("d/m/Y");

function bienvenida($user) {
  return "Hola, {$user}! 👋";
}

echo bienvenida($nombre);
echo "Hoy es: {$fecha}";
?>

TypeScript

JavaScript con tipado

Superconjunto de JavaScript creado por Microsoft que añade tipos estáticos. Hace el código más seguro y predecible. Muy usado en proyectos grandes con React o Angular.

🖥️ Lenguajes de Escritorio

Se usan para construir aplicaciones que se instalan directamente en tu computadora: editores de texto, juegos, software de diseño, sistemas operativos. Generalmente son lenguajes de más bajo nivel y alto rendimiento.

Java

Java

Orientado a objetos

"Write once, run anywhere". Java se ejecuta en la JVM (máquina virtual), lo que lo hace multiplataforma. Es el lenguaje base de Android y miles de aplicaciones empresariales.

Hola.java 
// Clase principal en Java
public class Hola {
  public static void main
      (String[] args) {

    String nombre = "Perla";
    System.out.println(
      "Hola, " + nombre
    );
  }
}
POOAndroidMultiplataforma
C#

C# (C Sharp)

Microsoft · .NET

Lenguaje de Microsoft para aplicaciones Windows, videojuegos (Unity) y servicios web con .NET. Combina la potencia de C++ con la simplicidad de Java.

C++

C++

Bajo nivel · Alto rendimiento

Extensión de C con orientación a objetos. Se usa en sistemas operativos, motores de videojuegos (Unreal), software embebido y aplicaciones que requieren máximo rendimiento.

🎨 Lenguajes de Diseño y Maquetación

Más allá de la programación, existen lenguajes y herramientas especializadas en la presentación visual, la maquetación de documentos y el diseño de interfaces. Son la intersección entre el arte y el código.

SASS

SASS / SCSS

Preprocesador CSS

Extiende CSS con variables, funciones, anidamiento y mixins. Permite escribir estilos más organizados y reutilizables. El código se "compila" a CSS estándar.

estilos.scss 
// Variables en SASS
$rosa: #e91e8c;
$lila: #9c6bc4;
$radio: 12px;

.card {
  border-radius: $radio;
  background: $lila;

  &:hover {
    background: $rosa;
  }
}
PreprocesadorVariablesMixins
SVG

SVG

Scalable Vector Graphics

Lenguaje XML para describir gráficos vectoriales. Los SVG se escalan sin perder calidad. Son ideales para íconos, logos, ilustraciones e infografías interactivas en la web.

</svg>
